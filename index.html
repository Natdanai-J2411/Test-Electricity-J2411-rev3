🛠️ วิธีเปิดระบบเชื่อมโยง Realtime (Google Apps Script Deployment Guide)

หากคุณต้องการอัปเดตข้อมูลบน Google Sheets แล้วให้วิ่งเข้าหน้าเว็บ HTML นี้ทันที หรือกดส่งข้อมูลจากฟอร์มนี้แล้วให้บันทึกลง Google Sheets ทันที ให้ดำเนินการดังนี้:

  1. เปิดไฟล์ Google Sheet ของคุณขึ้นมา ไปที่เมนู Extensions (ส่วนขยาย) > Apps Script
  2. คัดลอกโค้ด Google Apps Script ด้านล่างนี้ไปวางในหน้าต่างแก้ไข
  3. กดปุ่ม Deploy > New deployment เลือกประเภทเป็น Web app
  4. ตั้งค่าสิทธิ์ตรง Who has access ให้เป็น Anyone แล้วกด Deploy
  5. นำลิงก์ Web App URL ที่ได้ มาวางในช่องด้านบนสุดของหน้า Dashboard นี้เพื่อเชื่อมระบบ Realtime สมบูรณ์แบบ!

*คลิกเพื่อไฮไลต์โค้ดทั้งหมดแล้วสั่งคัดลอก
